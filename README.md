# mPong
A minimal two player pong game written in C++ using the SFML graphics library.

# Compile & Run (Linux)

Compile:
```
chmod +x build.sh;
./build.sh;
```

Run:
```
./main;
```

# How to play

Controls:

W -> Move left player up
S -> Move left player down

UP -> Move right player up
DOWN -> Move right player down

ESC -> Quit Game
